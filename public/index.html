Motivational Quotes API

Presenting to you, an API, which presents awesome random quotes in very specific and unique format. Thanks to worldofivo for letting me use the quotes. You may want to visit the instagram page, else you will miss some really creative content. Special Thanks to The Coding Train for motivation and learning.

About

Ther are 2 endpoints to this API:
  • GET /quote: this will fetch a random quote and will give a JSON in response to it.
    • Click me to TEST.
  • GET /quote/{NUMBER}: This will fetch {NUMBER} quotes, and outputs them in JSON as an array.
    • Click me to TEST.

Color Theme: Color theme is same as one of VS code theme available in market place, One Dark Theme.
